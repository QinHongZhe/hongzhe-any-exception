# AnyException

#### 介绍
定义异常和错误状态码的枚举

#### 软件架构
软件架构说明

立志涵盖Java开发中，业务的和非业务的常用状态码。

异常和状态码采用枚举方式设计，有更好的设计欢迎提ISSUES交流。

返回消息和自定义描述是作者自己理解产物，有更好的文字表达，也欢迎交流。

自定义描述原则上用英文表述。


#### 安装教程

1.  打成jar包或拷贝该类，开箱即用。


#### 设计公约

1. 若两种服务或者框架表达的意思一致，应当返回一个状态码，类型改为CORE
(例如：HTTP请求 200 和其他框架或者服务返回200  应当都理解为服务ok)
